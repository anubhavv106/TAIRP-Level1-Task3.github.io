
# Image Slider

This is a simple image slider with 3D effect.

## 🚀 Features

- Responsive slider.
- 3D Effect.
- Friendly UI.

## 🛠️ Usage

1. Clone this repository: 
```bash 
git clone https://github.com/anubhavv106/TAIRP-Level1-Task3.github.io
   ```
2. Open the `index.html` file in your browser.
